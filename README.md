# BookieDookie

**BookieDookie** — это удобное приложение для распознавания текста с изображений и поиска книг через Google Books API.

## Возможности
- **OCR**: Распознавание текста с изображений с использованием Tesseract OCR.
- **Google Books API**: Автоматический поиск книг по распознанному тексту (название, авторы, обложка).
- **Графический интерфейс**: Удобный и простой интерфейс с вкладками для поиска, просмотра результатов, истории и информации о программе.
- **История поиска**: Хранение данных о ранее найденных книгах.
- **Поддержка нескольких языков**: OCR работает на русском и английском языках.

## Установка
1. Убедитесь, что у вас установлен **Python 3.7+**.
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/BookieDookie.git
3. Перейдите в папку проекта
   cd BookieDookie
4. Установите зависимости:
   pip install -r requirements.txt
5. Установите Tesseract OCR
6. Укажите путь к Tesseract OCR в файле config.json.

## Использование
1. Запустите приложение:
   python -m AiP.BookieDookie
2. Загрузите изображение, содержащее текст.
3. Приложение выполнит распознавание текста и осуществит поиск книги.
